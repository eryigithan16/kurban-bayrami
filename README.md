# Bayram Sürprizi (GitHub Pages)

Bu repo tek sayfalık, mobil uyumlu bir “Kurban Bayramı” sürpriz sitesi.

## Kişiselleştirme

`app.js` içindeki `CONFIG` alanlarını değiştir:
- `herName`: kız arkadaşının adı / hitap
- `fromName`: senin adın / imzan
- `message`: ana mesaj

## GitHub Pages ile yayınlama

1. GitHub’da repo **Public** olsun (Pages ücretsiz kullanım için genelde gerekli).
2. Repo > **Settings** > **Pages**
3. **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: `main` / `(root)`
4. Kaydedince GitHub sana bir URL verecek. Onu kız arkadaşına gönder.

Not: İlk yayın 1–3 dakika sürebilir.
